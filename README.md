# Student Marks Predictor

A machine learning web app that predicts student marks based on:

- Study Hours
- Attendance
- Previous Marks
- Sleep Hours

## Tech Stack

- React.js
- Node.js
- Express.js
- Python
- Scikit-learn

## Workflow

React → Axios → Express API → Python ML Model → Prediction

## Run Locally

### Backend
```bash
cd backens
npm install
node server.js
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
